## Hello, This is a simple webchat system.

## 本系统采用 Python + Django + Django Channels + bootstrap + MySQL + Redis 搭建。

### 实现功能

 - 用户注册
 - 用户登录验证
 - 查看用户信息和修改密码
 - 显示所有在线用户和离线用户
 - 用户进入和离开聊天室系统会发送广播
 - 实时聊天无延时
